Tsugi for Java - Library Code

This is a Java API library compatible with the PHP Tsugi application (http://csev.github.io/tsugi/). It allows you to build portable Java servlets that make use of the various LTI standards through the use of the Tsugi APIs.

Tsugi-Java API Documentation

Open JavaDoc for the Tsugi-Java API in a new window.

Sample Servlet

A sample Java servlet that uses this library is available at (http://csev.github.io/tsugi-servlet/). This page has instructions for compiling the library, starting the servlet and testing the servlet with PHP Tsugi.
